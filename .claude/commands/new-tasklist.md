現在のブランチ名を確認し、新しいタスクリストファイルを作成する。

## 手順

1. `git branch --show-current` で現在のブランチ名を確認する
2. ブランチ名からタスクの内容を推測し、ユーザーに「以下の内容でタスクリストを作成します。よいですか？」と確認する
3. 今日の日付を `YYYYMMDD` 形式で取得する
4. `.claude/tasklists/YYYYMMDD-[ブランチ名のfeat/を除いた部分].md` を作成する
5. ユーザーと対話しながらタスク一覧を埋める

## ファイルのフォーマット

```markdown
# [作業タイトル]

## 作業概要

[今回の作業の目的・背景を1〜2文で記述]

## タスク

- [ ] タスク1
- [ ] タスク2
- [ ] タスク3

## 完了条件

- [ ] `pnpm check` がエラーなく通る
- [ ] `pnpm build` がエラーなく通る
- [ ] `pnpm preview` で表示を目視確認した
- [ ] `.claude/specs/` の該当ドキュメントを更新した
```

## 注意

- タスクは具体的に書く（「実装する」ではなく「`BlogCard.astro` を作成する」）
- 完了条件はPR作成前チェックリストと合わせる
