git の状態を読み取り、`progress.md` と `.claude/tasklists/` を最新の状態に同期する。

## 手順

1. `git log main --oneline -20` を実行してmainブランチの最新コミットを確認する
2. `git branch --show-current` で現在のブランチを確認する
3. `.claude/progress.md` を読み込む
4. `.claude/tasklists/` 内の該当ファイルを読み込む
5. git logの内容と照合して以下を更新する

## 更新ルール

### progress.md
- mainにマージ済みのものは `✅` に変更する
- 現在のブランチで作業中のものは `🔧` のまま維持する
- 「現在のブランチ・PR状況」テーブルを実態に合わせて更新する
- 「最終更新」の日付を今日の日付に更新する
- 「次に着手する作業」を現状に合わせて更新する

### tasklists/
- マージ済みのコミット内容に対応するタスクのチェックボックスを `- [x]` に更新する
